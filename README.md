# NEP AI Timetable

## Project Overview

The **NEP AI Timetable Generator** is a web-based system designed to automatically generate academic timetables for educational institutions following the **National Education Policy (NEP)** guidelines.

The system uses **Artificial Intelligence techniques, specifically the Genetic Algorithm**, to create optimized timetables while minimizing scheduling conflicts between faculty members, courses, rooms, and time slots.

GitHub Repository:
https://github.com/Aslesha-git/NEP-AI-Timetable

---

## Objectives

* To automate the process of timetable generation.
* To reduce manual scheduling work.
* To optimize time slot allocation using AI techniques.
* To manage courses, faculty, rooms, and programs efficiently.

---

## Technologies Used

The project is developed using the following technologies:

* **React.js** – Used to build the interactive user interface.
* **Artificial Intelligence (Genetic Algorithm)** – Used to optimize timetable scheduling.
* **MongoDB** – Used as the database to store faculty, course, and timetable data.
* **CSS** – Used for designing and styling the application.

---

## Key Features

* Automatic timetable generation
* AI-based scheduling optimization
* Faculty and course management
* Room and program allocation
* Conflict-free timetable creation
* User-friendly interface

---

## System Modules

1. **Faculty Module**
   Used to add, edit, and manage faculty details.

2. **Room Module**
   Manages classroom and laboratory room information.

3. **Program Module**
   Handles academic programs and semester details.

4. **Course Module**
   Stores course subjects and related information.

5. **Generate Module**
   Uses the Genetic Algorithm to generate an optimized timetable.

6. **Timetable Module**
   Displays the generated timetable in an organized format.

7. **Settings Module**
   Allows configuration of system parameters and preferences.
